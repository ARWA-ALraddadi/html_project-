Coffee Grove Madinah Application

The Coffee Grove Madinah has a web form that allows users to order roasted coffee beans.
The back-end service is running at http://CoffeeGroveMadinah.com/order and it accepts GET request with the following parameters:

  • name: this parameter is to specify full name
  • phone: this parameter is to specify contact phone number
  • address: this parameter is to specify home address
  • td : this parameter is to specify type of address
    • res: Residend type of address
    • bus: Business type of address
  • cofbn: this parameter is to specify coffee Beans Type
    • cl: Colombian coffee
    • eth: Ethiopian coffee
    • ken: Kenya Coffee
    • arb: Arabica coffee
  • cofp: this parameter is to specify coffee Beans processed
    • na: Natural Processing
    • wash: Washed Processing
    • hon: Honey Processing
  • wt: this parameter is to specify coffee wight
    • one: 250 grams
    • two: 500 grams
    • thr: one kilograms
  • grd: this parameter is to specify Grind Type
    • cr: Coarse Grind
    • mc: Medium-Coarse Grind
    • med: Medium Grind
    • fin: Fine Grind
  • mgift: this parameter to write a short Gift Message
